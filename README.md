# BOKSklapps

Boxing Cardio App

## What is it?

User can choose timers, punches and tempo and the app will provide a workout.
BOKSklapps can be used on a boxing bag (recommended) or as shadow boxing app.

### What's so special about the code?

App uses flutter_riverpod as state manager for almost everything. Theming, timers,
lists of punches, it's all being stored and altered in Riverpod Providers.

App uses flex_color_scheme as theming manager for literally everything.
One-stop-shop for light/dark and everything in between.

App uses flutter_animate for all animations. So if you're interested, have a look!
